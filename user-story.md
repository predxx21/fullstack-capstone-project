# Historias de Usuario - FullStack Capstone Project

## Formato de Historia de Usuario
**Como** [tipo de usuario]  
**Quiero** [realizar una acción]  
**Para** [obtener un beneficio]

## Historias de Usuario

### Historia 1: Visualizar Catálogo
**Como** usuario visitante  
**Quiero** ver todos los regalos disponibles  
**Para** explorar las opciones de regalo

**Criterios de Aceptación:**
- [x] La página principal muestra una cuadrícula de regalos
- [x] Cada regalo muestra imagen, nombre y precio
- [x] Hay un máximo de 12 regalos por página

### Historia 2: Ver Detalles de Regalo
**Como** usuario interesado  
**Quiero** hacer clic en un regalo para ver sus detalles  
**Para** obtener más información antes de decidir

**Criterios de Aceptación:**
- [x] Al hacer clic se muestra página de detalles
- [x] Se muestra descripción completa del producto
- [x] Se muestran imágenes adicionales

### Historia 3: Buscar Regalos
**Como** usuario  
**Quiero** buscar regalos por categoría  
**Para** encontrar rápidamente lo que necesito

**Criterios de Aceptación:**
- [x] Hay una barra de búsqueda en la navegación
- [x] Los resultados se filtran en tiempo real
- [x] Se muestran mensajes si no hay resultados

### Historia 4: Registro de Usuario
**Como** visitante  
**Quiero** crear una cuenta  
**Para** guardar mis regalos favoritos

**Criterios de Aceptación:**
- [x] Formulario con nombre, email y contraseña
- [x] Validación de campos obligatorios
- [x] Confirmación por email

### Historia 5: Inicio de Sesión
**Como** usuario registrado  
**Quiero** iniciar sesión en mi cuenta  
**Para** acceder a mis funciones personalizadas

**Criterios de Aceptación:**
- [x] Formulario de login con email y contraseña
- [x] Manejo de errores de autenticación
- [x] Redirección después del login exitoso

### Historia 6: Perfil de Usuario
**Como** usuario autenticado  
**Quiero** ver y editar mi perfil  
**Para** mantener mi información actualizada

**Criterios de Aceptación:**
- [x] Visualización de información personal
- [x] Formulario para editar datos
- [x] Actualización exitosa con feedback visual

### Historia 7: Búsqueda Avanzada
**Como** usuario  
**Quiero** filtrar regalos por múltiples categorías  
**Para** encontrar opciones más específicas

**Criterios de Aceptación:**
- [x] Filtros por precio, categoría, popularidad

### Historia 8: Favoritos
**Como** usuario autenticado

**Criterios de Aceptación:**
- [x] Botón de favoritos en cada producto

### Historia 9: Recomendaciones

**Criterios de Aceptación:**
- [x] Recomendaciones basadas en visitas
